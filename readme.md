## âš™ï¸ Installation & Setup

<details>
   <summary><b>Clone the repository</b></summary>
  
   ```bash
   git clone https://github.com/Saquib-Anjum/quick-carz.git
   cd quick-carz
   ```
</details>

<details>
   <summary><b>Install npm Dependencies</b></summary>
  
   <b>Backend</b>
   ```bash
   cd backend
   npm install
   ```
   <b>Frontend</b>
   ```bash
   cd ../frontend
   npm install
   ```
</details>

<details>
   <summary><b>Set up environment variables</b></summary>
  
   ```env
   PORT=5000
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   IMAGEKIT_PUBLIC_KEY=your_public_key
   IMAGEKIT_PRIVATE_KEY=your_private_key
   IMAGEKIT_URL_ENDPOINT=your_url_endpoint
   ```
</details>
